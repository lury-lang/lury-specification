## 7. トップレベル要素 トップレベル要素は、Lury の文法で最も上位に用いられる文法です。 ### 7.1. ファイル入力 ```bnf program : | program_lines program_lines : program_line | program_lines program_line program_line : statement | NewLine ``` 文法 `program` は Lury プログラムが記述されたファイルを読み込む際に最初に用いられる文法です。ファイル入力の場合、全ての文法は 文法 `program` に入れ子となります。 空のファイルを受け付けるために、`program` は唯一 _空_ (くう) の状態に遷移することを許されています。_空_ は 0 バイトの入力を意味します。また、改行だけが記述されたプログラムも、文法的には正しい入力とみなされます。 まとめると、文法的に正しい Lury プログラムは、文（単純文と複合文）と改行 （空行）が 0 回以上繰り返したものとなります。 ### 参考文献 - [9. トップレベル要素 — Python 3.4.3 ドキュメント](http://docs.python.jp/3.4/reference/toplevel_components.html)
